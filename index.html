How to Install

Follow these steps to install Nega Hack using Geode:

  1. Install the Geode SDK from https://geode-sdk.org.
  2. Download the Geode mod file.
  3. Locate your game install folder:
    • If you bought the game on Steam, open your library, right-click on Geometry Dash > Manage > Browse Local Files.
    • If you have a cracked version, just find where GeometryDash.exe is located.
  4. Open the geode folder, then go to mods.
  5. Drag teamtuff.negahack.geode into the mods folder.
  6. Start or restart the game.
  7. On the first launch, you will be asked to log in. You can ignore this and simply press the "Login" button.
  8. Congratulations! You just saved $5 by not buying Mega Hack.
